# InsightFlow Project Agents

## Project Preferences
- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS (Strictly no Vanilla CSS)
- **Backend:** FastAPI (Python 3.13)
- **AI Engine:** Gemini 3.1 Pro (use thinking_level: high)

## The Architect (Lead Agent)
- **Role:** Design the data schemas and system architecture.
- **Rule:** Always use Pydantic for data validation.
- **Rule:** Every insight must be "Actionable" (Imperative verbs only).

## The Frontend Wizard
- **Role:** Build the dashboard and UI.
- **Rule:** Use a "Bento Box" layout with glassmorphism effects.
- **Rule:** Use Shadcn/UI components for all tables and cards.
