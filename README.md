## 📄 Document-to-Action Engine

Upload any document — resume, contract, or report — 
and get structured insights with clear next steps, 
recommendations, and decisions powered by Gemini AI.

No more reading walls of text. Just upload and act.

### ✨ Features
- PDF/document upload and parsing
- AI-powered key information extraction
- Actionable next steps generation
- Supports resumes, contracts, reports and other documents
- Clean exportable output
- Supports various formats like PDFs, word files, .txt files and images
- Multiple file uploads for comparing documents like contracts from 2 different parties
- Translates the document analysis to Hindi or Marathi

### 🛠️ Built With
- Front-End: Next.Js, Vanilla CSS 
- AI models used: Gemini 2.5 Flash, Ollama qwen-2.5:7B (running locally)
- Back-End: Python, FastAPI
- OCR used: Tesseract OCR
